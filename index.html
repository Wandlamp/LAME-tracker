LAME Tracker instellen

Verbind de tracker met je eigen gratis Supabase database zodat alle data bewaard blijft. Dit doe je eenmalig in 5 minuten.

1
Maak een gratis account op supabase.com en maak een nieuw project aan.
2
Ga naar SQL Editor in je project en voer de SQL uit die onderaan staat (kopieer en plak).
3
Ga naar Project Settings → API. Kopieer je Project URL en anon public key hieronder.

        Hoe werkt de berekening?

        We gebruiken de hypergeometrische verdeling: de exacte kans om precies k kaarten van een bepaald type te trekken uit een deck zonder teruglegging.


        Volledige mulligan: Alle 7 terug, opnieuw 7. P(≥1) = 1 − P(0)²

        Gedeeltelijke mulligan: X vastgehouden, (7−X) nieuwe kaarten uit resterende (60−X).
